# **App Name**: InspireMe

## Core Features:

- Quote Generation: Generate an inspirational quote based on user-selected themes.
- Theme Selection: Allow users to select themes like 'success', 'motivation', or 'perseverance' to tailor the quote generation.
- Quote Display: Display quotes in a visually appealing and readable format, using dynamic typography.
- AI Quote Customization Tool: Allow users to input a personal detail (e.g., 'a challenge I'm facing') which is then incorporated as a tool for the AI to generate a personalized quote.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey wisdom and depth.
- Background color: Very light lavender (#F0E6FF), nearly white, to provide a calming backdrop.
- Accent color: Muted violet (#8A2BE2) to highlight key elements and add visual interest.
- Headline font: 'Playfair', a modern serif to capture an elegant, fashionable feel. Body font: 'PT Sans' for a clear, readable body.
- Use simple, line-based icons to represent different quote themes.
- Use subtle fade-in animations when quotes are displayed to add a sense of reflection.
